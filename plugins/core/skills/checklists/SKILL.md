# Workflow checklists

Five phase checklists, one file each, in this skill's directory. Read ONLY the
file for the phase at hand: plan.md, review.md, test.md, security.md, handoff.md.
Work through its items explicitly — name each item as done, skipped (say why), or
blocked. These are judgment aids, not bureaucracy: an item that doesn't apply is
marked n/a with one clause of reasoning, never silently dropped.
