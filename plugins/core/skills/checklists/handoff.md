# Session Handoff Checklist

## 1. Session Summary
- What tasks were completed?
- What decisions were made and why?
- What's currently in progress?
- What blockers or issues exist?

## 2. Git Status
- Commit any uncommitted changes (ask user first)
- Note any WIP work

## 3. Update the project's progress/architecture docs if the repo keeps them.

## 4. Update Documentation (if needed)
- `docs/ARCHITECTURE.md` — architecture decisions
- `docs/MIGRATION.md` — migration approach changes
- `docs/superpowers/specs/[feature]-design.md` — feature specs

## 5. Handoff Summary Template

```
### Completed This Session
- [bullet points]

### Current State
[brief description]

### Next Steps
1. [priority 1]
2. [priority 2]
3. [priority 3]

### Key Decisions Made
- [decision]: [rationale]

### Open Questions/Blockers
- [if any]
```
